============================================================
0. 文稿定位
============================================================

第四编 R1.0 不直接构造粒子谱，也不直接宣称得到连续场论。

第四编 R1.0 只做一件事：

    在第三编已经通过稳定性与粗粒化验证的前提下，
    在块级离散几何承载层上定义场、算符与作用量接口。

第二编交付事件、严格因果偏序、因果切片、更新痕迹与干涉痕迹；
第三编把这些对象转化为稳定的事件级空间承载图与块级空间-因果结构；
第四编才允许在这些承载结构上定义场。

核心链条为：

    第二编：
    (𝔈, ≺, 𝒰(e), ⋈_C, ℐ(e))

    ↓

    第三编：
    G_Σ, Π_Σ, G_B, d_B, Vol_B, D_B, ≺_B

    ↓

    第四编：
    Φ : Π → V
    Δ_B
    S_field
    S_upd
    S_int
    S_eff

第四编的基本原则：

    场不是定义在连续点 x 上；
    场首先定义在事件、切片或块上；
    正式主线优先使用块级场 Φ:Π→V。

============================================================
2. 第四编红线
============================================================

第四编不得直接引入：

    连续流形 M；
    预设坐标 x^μ；
    背景度规 g_{μν}；
    连续导数 ∂_μ；
    连续积分 ∫d^dx；
    外部时间参数 t；
    先验粒子谱；
    先验洛伦兹不变性。

第四编允许引入：

    块级场 Φ(B)；
    图上的差分；
    图拉普拉斯；
    块级体积权重；
    块级空间邻接；
    块级因果边；
    离散作用量；
    离散变分方程；
    谱稳定性检查。

裁决原则：

    连续场论只能作为后续极限检验对象，
    不能作为第四编的起点。

============================================================
3. 场的承载域
============================================================

第四编保留三种场定义层级。

层级 A：事件级场。

    φ : 𝔈 → V

其中 V 可以是 ℝ、ℂ、ℝ^n 或其他内部空间。

层级 B：切片级场。

    φ_Σ : Σ → V

其中 Σ 是第三编给出的极大反链切片。

层级 C：块级场。

    Φ : Π → V

其中 Π 是第三编粗粒化后的块集合。

正式主线选择层级 C：

    Φ : Π → V

理由：

1. 事件级结构可能过细；
2. 切片级结构可能依赖切片选取；
3. 块级结构经过第三编稳定性与粗粒化检验；
4. 块级体积 Vol_B 可自然进入作用量。

默认最小场为实标量场：

    Φ : Π → ℝ

更高结构如复场、向量场、规范场、费米子场暂不作为第四编 R1.0 的主线对象。

============================================================
4. 块级体积内积
============================================================

定义块级体积权重：

    V_a = Vol_B(B_a)

对两个块级实场 Φ,Ψ:Π→ℝ，定义体积加权内积：

    ⟨Φ,Ψ⟩_B
    =
    Σ_{B_a∈Π} V_a Φ_a Ψ_a

其中：

    Φ_a = Φ(B_a)

由此定义场范数：

    ||Φ||_B^2
    =
    Σ_a V_a Φ_a^2

裁决原则：

    体积权重来自第三编的块级计数体积，
    不是连续测度 √|g|d^dx。

============================================================
5. 块级空间图与边权
============================================================

第三编交付块级空间图：

    G_B = (Π, ~_B)

其中：

    B_a ~_B B_b

表示两个块在同一切片或同一空间承载层上相邻。

定义空间边集合：

    E_s
    =
    { (a,b) : B_a ~_B B_b }

为避免重复计数，默认取无向边：

    a<b

定义边权：

    W^s_{ab} ≥ 0

可选取法：

1. 密度式权重：

    W^s_{ab} = dens(B_a,B_b)

2. 相似度平均权重：

    W^s_{ab}
    =
    Avg_{e∈B_a,q∈B_b} S_{Σ,r}(e,q)

3. 单位权重：

    W^s_{ab}=1

正式主线建议：

    若第三编已经使用密度式块邻接，
    则第四编使用 W^s_{ab}=dens(B_a,B_b)。

这样权重直接继承第三编的邻接来源。

============================================================
6. 块级空间图拉普拉斯
============================================================

定义空间图拉普拉斯：

    (Δ_s Φ)_a
    =
    1/V_a · Σ_{b:a~_B b} W^s_{ab}(Φ_b-Φ_a)

该定义满足：

    ⟨Φ, -Δ_s Φ⟩_B
    =
    1/2 Σ_{a~b} W^s_{ab}(Φ_a-Φ_b)^2

因此：

    -Δ_s

是非负的空间刚度算符。

定义空间梯度能：

    E_s[Φ]
    =
    1/2 Σ_{a~b} W^s_{ab}(Φ_a-Φ_b)^2

裁决原则：

    Δ_s 是图算符；
    它不是连续拉普拉斯算符；
    只有在后续连续极限中才可检验其是否近似 ∇²。

============================================================
7. 块级因果关系与因果覆盖边
============================================================

第三编交付块级严格因果关系：

    B_a ≺_B B_b

但若 ≺_B 已经是传递闭包，就不能直接把所有 ≺_B 边作为动力学相邻边。

第四编需要块级覆盖关系：

    B_a ⋖_B B_b

定义：

    B_a ⋖_B B_b

当且仅当：

    B_a ≺_B B_b

且不存在 B_q 使得：

    B_a ≺_B B_q ≺_B B_b

定义因果覆盖边集合：

    E_c
    =
    { (a,b) : B_a ⋖_B B_b }

其中 E_c 是有向边集合。

裁决原则：

    场的因果差分优先沿 ⋖_B 定义，
    而不是沿完整传递闭包 ≺_B 定义。

============================================================
8. 因果差分算符
============================================================

对因果覆盖边：

    B_a ⋖_B B_b

定义标量场的因果差分：

    D_c Φ(a,b)
    =
    Φ_b - Φ_a

若后续引入内部空间联络或规范结构，可改成：

    D_c Φ(a,b)
    =
    Φ_b - T_{ba}Φ_a

其中 T_{ba} 是从 B_a 到 B_b 的内部输运算符。

第四编 R1.0 的最小主线取：

    T_{ba}=1

因此：

    D_c Φ(a,b)=Φ_b-Φ_a

定义因果边权：

    W^c_{ab} ≥ 0

可选取法：

1. 因果保持率局部权重；
2. 事件级因果边密度权重；
3. 单位权重。

默认可取：

    W^c_{ab}=ratio_≺(B_a,B_b)

若第三编未提供局部 ratio_≺，则取 W^c_{ab}=1。

============================================================
9. 因果动能项
============================================================

定义因果动能：

    E_c[Φ]
    =
    1/2 Σ_{(a,b)∈E_c} W^c_{ab}(Φ_b-Φ_a)^2

这是沿块级因果覆盖边的离散差分能。

注意：

    E_c 不是普通欧氏空间梯度；
    它记录场沿因果方向的变化。

若要模拟洛伦兹型作用量，空间项和因果项的符号不能简单相同。

因此第四编保留两种形式。

============================================================
10. 稳定型作用量与洛伦兹型作用量
============================================================

形式 A：稳定型离散泛函。

    S_stable[Φ]
    =
    E_c[Φ]
    +
    c_s^2 E_s[Φ]
    +
    Σ_a V_a V_pot(Φ_a)

该形式适合数值稳定性、谱分析和有效能量检查。

形式 B：洛伦兹型离散作用量候选。

    S_L[Φ]
    =
    E_c[Φ]
    -
    c_s^2 E_s[Φ]
    -
    Σ_a V_a V_pot(Φ_a)

其中：

    c_s^2

是空间项相对于因果项的有效系数。

裁决原则：

    S_stable 用于验证；
    S_L 才是洛伦兹型场论候选；
    二者不得混淆。

第四编 R1.0 可以同时保留二者，但正式物理解释必须等连续极限与谱分析之后再给出。

============================================================
11. 最小标量场作用量
============================================================

取势能：

    V_pot(Φ)
    =
    1/2 m^2 Φ^2
    +
    λ/4 Φ^4

则稳定型标量场泛函为：

    S_stable[Φ]
    =
    1/2 Σ_{(a,b)∈E_c} W^c_{ab}(Φ_b-Φ_a)^2
    +
    c_s^2/2 Σ_{a~b} W^s_{ab}(Φ_a-Φ_b)^2
    +
    Σ_a V_a [
        1/2 m^2 Φ_a^2
        +
        λ/4 Φ_a^4
      ]

洛伦兹型候选为：

    S_L[Φ]
    =
    1/2 Σ_{(a,b)∈E_c} W^c_{ab}(Φ_b-Φ_a)^2
    -
    c_s^2/2 Σ_{a~b} W^s_{ab}(Φ_a-Φ_b)^2
    -
    Σ_a V_a [
        1/2 m^2 Φ_a^2
        +
        λ/4 Φ_a^4
      ]

此处：

    m, λ, c_s

是有效参数，不是从连续理论直接搬来的常数。

============================================================
12. 离散变分方程
============================================================

对稳定型二次场作用量：

    S_2[Φ]
    =
    1/2 Σ_{(a,b)∈E_c} W^c_{ab}(Φ_b-Φ_a)^2
    +
    c_s^2/2 Σ_{a~b} W^s_{ab}(Φ_a-Φ_b)^2
    +
    1/2 m^2 Σ_a V_a Φ_a^2

对 Φ_a 变分得到：

    Σ_{b:(a,b)∈E_c} W^c_{ab}(Φ_a-Φ_b)
    +
    Σ_{p:(p,a)∈E_c} W^c_{pa}(Φ_a-Φ_p)
    +
    c_s^2 Σ_{b:a~b} W^s_{ab}(Φ_a-Φ_b)
    +
    m^2 V_a Φ_a
    =
    0

若有源项 J_a，则右侧变为：

    V_a J_a

写成算符形式：

    K_B Φ = V J

其中：

    K_B

是由因果覆盖边、空间边与质量项共同定义的块级离散场算符。

============================================================
13. 块级传播子接口
============================================================

若 K_B 可逆或在适当边界条件下可伪逆，则定义块级 Green 函数：

    G_B^{field}(a,b)

满足：

    Σ_q K_{a q} G_B^{field}(q,b)
    =
    δ_{ab}

或体积加权版本：

    Σ_q K_{a q} G_B^{field}(q,b)
    =
    δ_{ab}/V_a

第四编 R1.0 只定义传播子接口，不把传播子解释为粒子。

裁决原则：

    粒子概念需要谱稳定性、近似对称性和可解释的能量-动量结构；
    第四编 R1.0 只到场算符与传播子接口为止。

============================================================
14. 更新痕迹的块级汇聚
============================================================

第二编交付事件级更新痕迹：

    𝒰(e)

第三编给出块划分：

    B_a ⊂ Σ 或 B_a ⊂ 𝔈_window

定义块级更新痕迹：

    𝒰(B_a)
    =
    ⋃_{e∈B_a} 𝒰(e)

若需要保留 multiplicity，可定义多重集合版本：

    𝒰_mult(B_a)
    =
    ⨄_{e∈B_a} 𝒰(e)

默认正式主线保留两种对象：

    𝒰_set(B_a)   去重集合；
    𝒰_mult(B_a)  多重集合。

其中：

    𝒰_set

用于结构存在性判断；

    𝒰_mult

用于计数、强度和作用量权重。

============================================================
15. 干涉痕迹的块级汇聚
============================================================

第二编交付事件级干涉痕迹：

    ℐ(e)= {q∈𝔈 : e ⋈_C q}

定义块级干涉痕迹：

    ℐ(B_a)
    =
    ⋃_{e∈B_a} ℐ(e)

定义块间干涉强度：

    I_{ab}
    =
    |{(e,q): e∈B_a, q∈B_b, e ⋈_C q}|
    /
    (|B_a||B_b|)

若 B_a 或 B_b 为空，则 I_{ab} 不定义；但合法块划分中不允许空块。

定义块级干涉关系：

    B_a ⋈_B B_b
    ⇔
    I_{ab} ≥ η_I

其中 η_I 是块级干涉阈值。

裁决原则：

    ⋈_B 必须从事件级 ⋈_C 汇聚；
    不得在第四编凭空定义新的相互作用关系。

============================================================
16. 更新作用量 S_upd
============================================================

第二编 R1.0 已把 𝒰(e) 作为第三编 S_upd 的来源。第四编 R1.0 在块级定义：

    S_upd
    =
    Σ_{B_a∈Π}
    L_upd(
      𝒰_mult(B_a),
      R(B_a),
      W(B_a),
      X(B_a)
    )

其中：

    R(B_a)=⋃_{e∈B_a}R_e
    W(B_a)=⋃_{e∈B_a}W_e
    X(B_a)=⋃_{e∈B_a}X_e

一个最小计数型候选为：

    L_upd(B_a)
    =
    α_U |𝒰_mult(B_a)|
    +
    α_R |R(B_a)|
    +
    α_W |W(B_a)|
    +
    α_X |X(B_a)|

更稳定的归一化形式为：

    L_upd(B_a)
    =
    α_U |𝒰_mult(B_a)|/Vol(B_a)
    +
    α_R |R(B_a)|/Vol(B_a)
    +
    α_W |W(B_a)|/Vol(B_a)
    +
    α_X |X(B_a)|/Vol(B_a)

裁决原则：

    S_upd 是更新结构对有效动力学的贡献；
    它不是连续几何曲率项；
    也不是外加物质势能项。

============================================================
17. 干涉作用量 S_int
============================================================

定义块内干涉项：

    S_int^local
    =
    Σ_{B_a}
    L_int(ℐ(B_a))

最小候选：

    L_int(ℐ(B_a))
    =
    β_I |ℐ(B_a)|/Vol(B_a)

定义块间干涉项：

    S_int^edge
    =
    Σ_{B_a ⋈_B B_b}
    K_int(B_a,B_b)

最小候选：

    K_int(B_a,B_b)
    =
    β_B I_{ab}

于是：

    S_int
    =
    S_int^local + S_int^edge

若干涉关系用于产生场源项，也可定义：

    J_a^{int}
    =
    j_0 · |ℐ(B_a)|/Vol(B_a)

并进入场方程：

    K_B Φ = V(J^{upd}+J^{int})

裁决原则：

    S_int 必须可追溯到第二编 ⋈_C 与 ℐ(e)；
    不得凭“相互作用”这个词直接加入任意耦合。

============================================================
18. 几何稳定性约束项 S_geom
============================================================

第三编已经定义稳定性指标：

    Stab_slice
    Stab_param
    Stab_coarse
    D_B
    Rough
    Preserve_≺

第四编可以把这些指标作为几何选择泛函，但必须区分：

    物理作用量项
    与
    结构筛选/约束项。

定义：

    S_geom
    =
    λ_D Var(D_B)
    +
    λ_R Rough_B
    +
    λ_P (1-Preserve_≺)
    +
    λ_S (1-Stab_slice)
    +
    λ_K (1-Stab_coarse)

此项用于筛选稳定几何候选。

裁决原则：

    S_geom 是几何稳定性约束；
    不应直接等同于爱因斯坦-希尔伯特作用量；
    后续若要得到曲率作用量，需要另行从粗粒化几何中推导曲率代理。

============================================================
19. 总有效泛函
============================================================

第四编 R1.0 的总有效泛函可写为：

    S_eff[Φ;𝔠_3]
    =
    S_field[Φ]
    +
    g_U S_upd
    +
    g_I S_int
    +
    g_G S_geom
    +
    S_constraint

其中：

    S_field      块级场作用量；
    S_upd        更新痕迹作用量；
    S_int        干涉痕迹作用量；
    S_geom       几何稳定性约束；
    S_constraint 红线约束项，如无环、局部有限、标签无关。

约束项可写为：

    S_constraint
    =
    Λ_cycle CyclePenalty(≺_B)
    +
    Λ_label LabelDependencePenalty
    +
    Λ_dense DenseFailurePenalty
    +
    Λ_sparse SparseFailurePenalty

其中 Λ 参数应视为硬约束极限：

    Λ → ∞

裁决原则：

    S_eff 是离散有效泛函；
    不是最终连续作用量；
    它的物理解释依赖后续极限、谱稳定性与实验/现象约束。

============================================================
20. 局域性条件
============================================================

第四编必须检查作用量是否局域。

局域性要求：

1. 空间项只沿 G_B 的边相互作用：

    B_a ~_B B_b

2. 因果项只沿 ⋖_B 覆盖边相互作用：

    B_a ⋖_B B_b

3. 干涉项只沿 ⋈_B 或局域干涉强度边相互作用；

4. 权重不得引入全局标签或远程任意配对。

若存在长程项：

    B_a 与 B_b 距离很远却强耦合

则必须给出来源：

    真实的因果覆盖；
    真实的事件级干涉；
    或粗粒化诱导的有效长程边。

否则该项不通过。

============================================================
21. 自伴性与谱稳定
============================================================

对空间拉普拉斯 Δ_s，应满足体积内积下的自伴性：

    ⟨Φ, Δ_s Ψ⟩_B
    =
    ⟨Δ_s Φ, Ψ⟩_B

在使用：

    (Δ_s Φ)_a
    =
    1/V_a Σ_b W^s_{ab}(Φ_b-Φ_a)

且 W^s_{ab}=W^s_{ba} 时成立。

场算符：

    K_B

必须进行谱检查：

1. 是否存在异常负模；
2. 零模是否可解释；
3. 小特征值是否对切片、参数和粗粒化稳定；
4. 谱维数是否有平台。

定义谱维数代理可通过热核迹：

    P(τ)
    =
    1/N Tr exp(-τ L_B)

    d_spec(τ)
    =
    -2 d log P(τ) / d log τ

其中 L_B 可取空间图拉普拉斯或稳定型场算符的几何部分。

裁决原则：

    稳定谱是进入粒子或连续场论的前置条件。

22. 源项：更新源与干涉源
============================================================

定义更新源：

    J_a^{upd}
    =
    j_U · |𝒰_mult(B_a)|/Vol(B_a)

定义拒绝/冲突源：

    J_a^{X}
    =
    j_X · |X(B_a)|/Vol(B_a)

定义干涉源：

    J_a^{int}
    =
    j_I · |ℐ(B_a)|/Vol(B_a)

总源项：

    J_a
    =
    J_a^{upd}
    +
    J_a^{X}
    +
    J_a^{int}

场方程：

    K_B Φ = V J

其中：

    (VJ)_a = V_a J_a

裁决原则：

    源项必须来自第二编事件痕迹；
    不得凭空指定外部物质分布。

23. 最小玩具例：一维链上的块级标量场
============================================================

沿用第三编 1+1 因果菱形格玩具例。

在某一切片上，粗粒化后得到一维块链：

    B_1 ~_B B_2 ~_B ... ~_B B_N

取：

    W^s_{a,a+1}=1
    V_a=1

空间拉普拉斯为：

    (Δ_s Φ)_a
    =
    Φ_{a+1}+Φ_{a-1}-2Φ_a

边界处按实际邻接减少项。

若相邻切片之间存在块级因果覆盖边：

    B_a^l ⋖_B B_a^{l+1}

则因果差分为：

    Φ_a^{l+1}-Φ_a^l

最小二次洛伦兹型离散作用量为：

    S_L
    =
    1/2 Σ_{l,a} (Φ_a^{l+1}-Φ_a^l)^2
    -
    c_s^2/2 Σ_{l,a} (Φ_{a+1}^l-Φ_a^l)^2
    -
    m^2/2 Σ_{l,a} (Φ_a^l)^2

离散方程形式为：

    Φ_a^{l+1}-2Φ_a^l+Φ_a^{l-1}
    -
    c_s^2(Φ_{a+1}^l-2Φ_a^l+Φ_{a-1}^l)
    +
    m^2 Φ_a^l
    =
    J_a^l

这只是玩具极限，不能反推连续坐标先验。

它的作用是验证：

    因果差分 + 空间图拉普拉斯

可以在规则结构上恢复熟悉的离散波/Klein-Gordon 型形式。

24. 失败模式
============================================================

F1. 几何承载失败。

    G_B 不稳定，D_B 无平台，或 Stab_coarse 很低。
    结果：第四编不能定义可靠场算符。

F2. 因果覆盖失败。

    ≺_B 存在环，或 ⋖_B 不可稳定定义。
    结果：因果动能项无效。

F3. 图拉普拉斯失败。

    Δ_s 不自伴，或权重依赖标签。
    结果：空间算符不合格。

F4. 作用量非局域失败。

    S_field 或 S_int 引入无来源长程耦合。
    结果：回退检查第三编邻接和第二编干涉关系。

F5. 谱不稳定失败。

    K_B 的低谱随切片、参数或粗粒化剧烈变化。
    结果：不能进入粒子谱或连续极限讨论。

F6. 痕迹脱源失败。

    S_upd、S_int 或 J_a 不能追溯到 𝒰(e)、ℐ(e)、X_e。
    结果：第四编偷渡了外部物质项。

F7. 连续偷渡失败。

    先写连续拉氏量再离散化，却声称无背景推导。
    结果：违反第四编红线。
