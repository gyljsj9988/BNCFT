## A3.0 v0.3 目标

A3 v0.3 的目标不是新增理论，而是把 A3 v0.2 的回填位转成可合并、可审稿、可继续实填的工程文本。当前可确认的冻结结构已经全部落地；当前缺少 A2 原文支撑的数值、公式和标签，不再以未解释占位符出现，而是转入“未决项清单”。

v0.3 的完成口径如下：

1. A3 的默认路径固定为 N=8。
2. Xi_8 向量、D_eff、区域分类与三态输出的接口保持不变。
3. 所有门控均遵守“任一回退即整体回退；无回退但任一未决即整体未决；全部通过才通过”。
4. 缺少 A2 原值的位置不得猜测，必须写明缺失内容、影响范围与封版动作。
5. v0.3 可以作为 A2 回填的施工稿，但不得标记为数值封版稿。

## A3.2 默认 N=8 模板

默认工程模式固定为：

```text
N := 8
K := K_8
```

v0.3 已实填的执行规则：

- 未显式授权例外时，所有 Xi_k、D_eff 与区域判定均以 N=8 为唯一默认模板。
- A3 不决定 K_8 的重新编号方式；若 A2 采用 k=0...7，则 A3 继承；若 A2 采用 k=1...8，则 A3 继承。
- 非 N=8 调用不得静默进入默认流程。
- 非 N=8 但可由 A2 授权例外解释时，状态为“未决”。
- 非 N=8 且与冻结接口冲突、不可修复时，状态为“回退”。

待 A2 回填项：

| 项 | 内容 | 当前状态 |
| --- | --- | --- |
| K_8 正式索引 | k 的起止、排序、槽位含义 | 未决：待 A2-N8 原文 |
| 例外模式 | 是否允许非 N=8；允许条件是什么 | 未决：待 A2-N8 原文 |
| 索引别名 | 是否允许槽1...槽8 与正式 k 映射 | 未决：待 A2-N8 原文 |

## A3.3 Xi_8 向量接口

A3 使用 Xi_k(α,β) 的 A2 冻结定义，不重新推导，不改归一化，不跳槽。

```text
Xi_8(α,β) := (Xi_k(α,β))_{k∈K_8}
```

v0.3 已实填的有效性规则：

1. 八槽均须可计算、有限、符合 A2 有效域与归一化约束。
2. 任一槽位缺失，不得用邻近槽位、均值、插值或人工判断替代。
3. 可补充、可复查、可由 A2 已定义规则修复的，状态为“未决”。
4. 不可定义、非有限、奇异、与冻结定义冲突且不可修复的，状态为“回退”。
5. Xi 向量只进入默认 D_eff 算法；不得绕过 D_eff 直接分类，除非 A2 区域树明文授权。

## A3.4 D_eff 默认算法接口

D_eff 只允许由 A2 默认算法得到：

```text
D_eff := D_default(Xi_8(α,β))
```

v0.3 已实填的默认算法纪律：

- 输入只能是完整且有效的 Xi_8 向量。
- 默认算法前置条件缺失但可复核时，状态为“未决”。
- 默认算法无法启动、前置条件冲突、输出非有限或触发强制回退时，状态为“回退”。
- 回退状态下，正式 D_eff 必须置为 ∅。
- 不允许经验拟合、人工补算、后处理修正或替代算法覆盖默认算法。

## A3.5 标准输出与状态合成

A3 标准输出保持四元组：

```text
O_A3 = (D_eff, R, S, E)
```

| 字段 | 含义 | 通过时 | 未决时 | 回退时 |
| --- | --- | --- | --- | --- |
| D_eff | 默认算法输出 | 正式数值 | 带 ? 的复核值或空值 | ∅ |
| R | 区域标签 | 唯一区域 | 候选集合或边界态 | ∅ |
| S | 状态 | 通过 | 未决 | 回退 |
| E | 解释项 | 记录通过门控 | 记录缺失、边界或候选 | 记录强制回退原因 |

状态合成规则固定为：

```text
if any(FALLBACK flag):
    S := 回退
elif any(PENDING flag):
    S := 未决
else:
    S := 通过
```

通过不是默认结果，而是所有必要门控均无未决、无回退之后的剩余结果。

## A3.8 通过 / 未决 / 回退纪律

### A3.8.1 通过

仅当以下条件全部满足时，允许输出“通过”：

1. N=8 与 K_8 已确认；
2. α、β 在 A2 有效域内；
3. Xi_8 八槽完整、可计算、有限、稳定；
4. D_eff 默认算法前置条件满足；
5. D_eff 输出满足 A2 有限性、范围与稳定性要求；
6. G0-G10 均无未决、无回退；
7. 区域树输出唯一 A2 冻结区域；
8. E 中记录触发路径、阈值来源和最终区域原因。

### A3.8.2 未决

出现以下任一情况，必须输出“未决”：

- A2 原值缺失但可补充；
- 输入有效域待确认；
- Xi 槽位可复查但尚未确认；
- D_eff 前置条件可补齐但尚未补齐；
- 阈值落入 A2 未决带；
- 区域树输出多个候选区域；
- 需要人工复核但尚未触发强制回退。

未决不得伪装为通过；候选区域不得压成唯一区域。

### A3.8.3 回退

出现以下任一情况，必须输出“回退”：

- 默认 N=8 模板无法成立，且无 A2 授权例外；
- 输入与 Xi_k 冻结定义冲突；
- 任一关键 Xi_k 不可定义且不可修复；
- D_eff 默认算法无法启动；
- D_eff 非有限、奇异或明显失稳；
- G0-G10 任一门触发 A2 强制回退；
- 区域树无合法出口；
- 需要替代算法才能继续，但 A2 未授权。

回退后不得输出正式 D_eff 或正式区域标签。

## A3.9 v0.3 默认伪代码

```text
A3_CLASSIFY_v0_3(α, β):

    N := 8
    K := K_8

    flags := []

    if K_8 index convention is not confirmed:
        flags.add(PENDING, "K_8 索引待 A2 回填")

    if not VALID_DOMAIN_A2(α, β):
        if DOMAIN_RECHECKABLE_A2(α, β):
            flags.add(PENDING, "参数有效域待确认")
        else:
            return (∅, ∅, 回退, "参数无效或定义冲突")

    Xi := COMPUTE_XI_8_A2(α, β)

    if Xi has missing slot:
        if REPAIRABLE_BY_A2_RULE(Xi):
            flags.add(PENDING, "Xi 槽位待补充或复查")
        else:
            return (∅, ∅, 回退, "Xi 槽位不可修复")

    if any Xi_k is non-finite or violates A2 exception rule:
        return (∅, ∅, 回退, "Xi 触发强制回退")

    D_eff := D_DEFAULT_A2(Xi)

    if D_eff default algorithm preconditions are pending:
        flags.add(PENDING, "D_eff 前置条件待确认")

    if D_eff default algorithm fails:
        return (∅, ∅, 回退, "D_eff 默认算法失败")

    gate_flags := APPLY_G0_G10_A2(Xi, D_eff)

    if gate_flags contains FALLBACK:
        return (∅, ∅, 回退, "触发 A2 强制回退门控")

    if flags contains PENDING or gate_flags contains PENDING:
        return (D_eff?, candidate_regions?, 未决, "存在未决门控或 A2 原值缺失")

    R := APPLY_REGION_TREE_A2(Xi, D_eff)

    if UNIQUE(R):
        return (D_eff, R, 通过, "区域唯一且 G0-G10 通过")

    if MULTIPLE_CANDIDATES(R):
        return (D_eff?, R_candidates, 未决, "区域不唯一")

    return (∅, ∅, 回退, "无合法区域出口")
```

| P8 | A2 额外证据字段 | 解释项 E | 报告字段可能不完整 | 回填 A2-EVIDENCE 要求 |

## A3.12 冻结一致性审计

| 审计项 | v0.3 通过条件 | 失败表现 | 处置 |
| --- | --- | --- | --- |
| N=8 | 默认只取 N=8；例外需 A2 授权 | 未授权切换 N | 未决或回退 |
| Xi 定义 | 只引用 A2-XI；八槽完整 | 重定义、漏槽、改归一化 | 停止封版 |
| D_eff | 只调用默认算法 | 替代算法、经验拟合、人工补算 | 回退 |
| 阈值表 | G0-G10 与 A2-TH 对应 | 阈值缺失、弱化、投票化 | 未决或回退 |
| 区域树 | 区域名和分支沿用 A2 | 新增、合并、重命名区域 | 停止封版 |
| 三态纪律 | 回退 > 未决 > 通过 | 未决写成通过；回退后继续分类 | 判定无效 |
| 报告证据 | E 记录门控、阈值来源、路径、原因 | 只有结论无证据 | 未决 |

## A3.13 v0.3 停止条件

以下任一情况存在时，A3 v0.3 不得升为“实填封版”：

1. 任一 Xi 槽位缺公式、有效域或异常规则。
2. D_eff 默认算法缺前置条件、步骤或失败路径。
3. G0-G10 任一门缺比较符、阈值或边界归属。
4. 区域树缺正式标签、分支条件或无出口规则。
5. 文稿出现未授权替代算法、经验修正或人工覆盖。
6. 有回退条件但仍继续输出正式 D_eff 或正式 R。
7. 有多个候选区域但被压成单一区域。

## A3.14 v0.3 结论

A3 v0.3 当前完成的是“实填施工版”：结构、状态纪律、门控接口和报告格式已经可执行；A2 原始数值、公式、区域标签与分支条件尚未进入本文，因此不能宣称为最终数值封版。

可合并结论：

```text
(α, β)
  -> N=8 / K_8
  -> Xi_8(α,β)
  -> D_eff = D_default(Xi_8)
  -> G0...G10
  -> A2 区域树
  -> O_A3 = (D_eff, R, S, E)
```

封版前唯一剩余动作：将 A2-N8、A2-XI、A2-DEFF、A2-TH、A2-REGION、A2-EVIDENCE 的原文逐项回填，并重新跑 A3.12 审计表。
