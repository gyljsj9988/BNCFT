3. 三层路径和
============================================================

第五编区分三种路径和，不能混用。

层级 I：固定块级几何上的场路径和。

给定一个已经通过验证的几何-因果接口包 𝔠_4，只对场配置 Φ 求和：

    Z_field[𝔠_4]
    =
    ∑_{Φ∈𝔽(Π)} Weight(Φ | 𝔠_4)

或连续值场时形式写作：

    Z_field[𝔠_4]
    =
    ∫ 𝒟_Π Φ · Weight(Φ | 𝔠_4)

其中 𝒟_Π Φ 是块集合 Π 上的离散场测度，不是连续流形上的场测度。

层级 II：固定因果事件结构上的几何-场路径和。

给定第二编事件结构：

    (𝔈,≺,𝒰,⋈_C,ℐ)

允许不同切片、邻接、粗粒化和场配置：

    Z_geom+field[𝔈,≺]
    =
    ∑_{G_B,Π,Φ}
    Weight(G_B,Π,Φ | 𝔈,≺,𝒰,ℐ)

层级 III：因果事件结构本身的路径和。

对可能的事件结构求和：

    Z_event
    =
    ∑_{[(𝔈,≺)]}
    ∑_{G_B,Π,Φ}
    Weight(𝔈,≺,G_B,Π,Φ)

其中：

    [(𝔈,≺)]

表示按标签同构类求和，而不是按事件编号求和。

裁决原则：

    固定几何场路径和最稳；
    几何-场路径和次之；
    因果事件结构路径和最强，也最危险；
    不能在没有测度和同构商的情况下直接写全路径和。

============================================================
4. 标签同构商
============================================================

若事件或块只是标签化对象，则不能把不同编号但结构同构的对象重复计数。

定义 4.1（结构同构）。

两个块级结构 𝔠_4 与 𝔠'_4 同构，当且仅当存在双射：

    π: Π → Π'

满足：

1. 空间邻接保持：

    B_a ~_B B_b
    ⇔
    π(B_a) ~'_B π(B_b)

2. 因果偏序保持：

    B_a ≺_B B_b
    ⇔
    π(B_a) ≺'_B π(B_b)

3. 覆盖关系保持：

    B_a ⋖_B B_b
    ⇔
    π(B_a) ⋖'_B π(B_b)

4. 体积、权重、痕迹强度在允许误差内保持：

    Vol_B(B_a) = Vol'_B(π(B_a))

    W^s_{ab}=W'^s_{π(a)π(b)}

    W^c_{ab}=W'^c_{π(a)π(b)}

5. 场配置对应：

    Φ_a = Φ'_{π(a)}

路径和应按同构类求和：

    ∑_{[𝔠_4]}

而不是：

    ∑_{labeled 𝔠_4}

裁决原则：

    标签不是物理自由度；
    标签重复计数必须消除或除以自同构群大小。

5. 自同构因子
============================================================

对于结构 𝔠_4，定义自同构群：

    Aut(𝔠_4)

若采用带标签求和再校正，则权重应包含：

    1 / |Aut(𝔠_4)|

于是可写：

    Z
    =
    ∑_{labeled 𝔠_4}
    1/|Aut(𝔠_4)|
    ∫𝒟_ΠΦ · Weight(Φ|𝔠_4)

但正式主线更推荐直接对同构类求和：

    Z
    =
    ∑_{[𝔠_4]}
    ∫𝒟_ΠΦ · Weight(Φ|𝔠_4)

因为这样更符合无背景红线。

============================================================
6. 权重的两种形式
============================================================

第四编区分了稳定型泛函与洛伦兹型候选作用量。

因此第五编也区分两类权重。

形式 A：稳定型权重。

    Weight_E(Φ|𝔠_4)
    =
    exp[-S_stable(Φ;𝔠_4)]

或：

    exp[-S_eff^E(Φ;𝔠_4)]

该形式适合：

    收敛性分析；
    数值采样；
    稳定性筛选；
    粗粒化流验证。

形式 B：洛伦兹型振幅。

    Amp_L(Φ|𝔠_4)
    =
    exp[i S_L(Φ;𝔠_4)]

或：

    exp[i S_eff^L(Φ;𝔠_4)]

该形式适合：

    相位干涉；
    量子振幅；
    因果路径和候选。

裁决原则：

    稳定型权重用于验证，不等同于物理量子振幅；
    洛伦兹型振幅是物理候选，但更难定义收敛；
    二者不得混写。

============================================================
7. 场测度
============================================================

对块级实标量场：

    Φ: Π → ℝ

若 Π 有 N 个块，则形式测度为：

    𝒟_Π Φ
    =
    ∏_{a=1}^N dΦ_a · μ_a

其中 μ_a 可以取：

1. 平坦测度：

    μ_a=1

2. 体积加权测度：

    μ_a=V_a^{1/2}

3. 归一化局域测度：

    μ_a=f(V_a,deg(B_a),Vol_B(B_a))

正式主线 R1.0 建议先使用平坦测度，并把体积权重放入作用量内积与质量项中。

若后续发现粗粒化下测度不稳定，再引入体积加权测度。

裁决原则：

    测度选择必须显式写出；
    不能把连续场论测度直接搬来。

8. 固定几何上的高斯场路径和
============================================================

若第四编给出二次作用量：

    S_2[Φ]
    =
    1/2 Φ^T K_B Φ
    -
    J^T V Φ

其中 V 是块级体积对角矩阵：

    V_{ab}=V_a δ_{ab}

则稳定型路径和为：

    Z_2[J|𝔠_4]
    =
    ∫ d^NΦ
    exp[
      -1/2 Φ^T K_B Φ
      + J^T V Φ
    ]

若 K_B 正定，则：

    Z_2[J|𝔠_4]
    =
    C · (det K_B)^(-1/2)
    · exp[
        1/2 J^T V K_B^{-1} V J
      ]

传播子为：

    G_B^{field}
    =
    K_B^{-1}

或体积加权版本：

    G_V
    =
    K_B^{-1} V

裁决原则：

    这是固定离散几何上的场传播；
    不能直接解释为粒子传播；
    粒子解释需要低谱稳定与近似对称性。

9. 非二次相互作用与扰动展开
============================================================

若包含：

    λ/4 Σ_a V_a Φ_a^4

则：

    S[Φ]
    =
    S_2[Φ] + S_int_field[Φ]

其中：

    S_int_field[Φ]
    =
    λ/4 Σ_a V_a Φ_a^4

稳定型路径和：

    Z[J]
    =
    ∫𝒟Φ exp[-S_2[Φ]-S_int_field[Φ]+J^TVΦ]

可形式展开为：

    Z[J]
    =
    exp[
      -S_int_field(δ/δJ)
    ] Z_2[J]

但第五编 R1.0 不把该扰动展开解释为连续费曼图，只将其作为离散场相关函数的生成接口。

定义 n 点函数：

    ⟨Φ_{a_1}...Φ_{a_n}⟩
    =
    1/Z[0] ·
    δ^n Z[J] /
    δJ_{a_1}...δJ_{a_n}
    |_{J=0}

若采用体积源项，则导数约定需包含 V_a。

10. 几何-场联合权重
============================================================

在固定因果事件结构下，第三编可能给出多个候选：

    G_B, Π, W^s, W^c

第四编为每个候选定义：

    S_eff[Φ;𝔠_4]

第五编可定义联合权重：

    Weight(G_B,Π,Φ | 𝔈,≺)
    =
    exp[
      -S_eff^E(Φ;𝔠_4)
    ]
    · M_geom(𝔠_4)

其中 M_geom 是几何测度因子。

最小选择：

    M_geom(𝔠_4)=1

更严格选择：

    M_geom(𝔠_4)
    =
    1/|Aut(𝔠_4)| · χ_pass(𝔠_4)

其中：

    χ_pass(𝔠_4)=1

当且仅当第三编和第四编通过条件满足；否则为 0。

也可使用软约束：

    M_geom(𝔠_4)
    =
    exp[-S_geom(𝔠_4)]

裁决原则：

    χ_pass 是硬筛选；
    exp[-S_geom] 是软权重；
    二者不能同时无说明地混合。

11. 因果事件结构权重
============================================================

对完整事件结构路径和，需要为：

    (𝔈,≺,𝒰,⋈_C,ℐ)

定义结构权重。

由于第二编中事件由：

    C → Ready(C) → ApplySet(C) → e

形成，事件结构权重应追溯到：

    Ready 成功；
    ApplySet 非空；
    冲突消解；
    N5 裁决；
    无环性；
    局部有限性；
    标签无关性。

定义硬约束函数：

    χ_event(𝔈,≺)
    =
    1

当且仅当：

1. (𝔈,≺) 是局部有限严格偏序；
2. 事件 e=(U_e,R_e,W_e,X_e) 均有限；
3. Ready/ApplySet 来源可追溯；
4. N5 与 Resolve 无标签依赖；
5. 无因果环；
6. 切片族可构造；
7. 第三编至少存在一个稳定几何承载候选。

否则：

    χ_event=0

定义事件结构权重候选：

    Weight_event(𝔈,≺)
    =
    χ_event(𝔈,≺)
    · exp[-S_event(𝔈,≺,𝒰,ℐ)]

其中 S_event 可由：

    S_upd + S_int + S_order + S_localfinite

组成。

第五编 R1.0 不固定 S_event 的最终形式，只规定它不得脱离第二编痕迹来源。

    Z_III

洛伦兹型版本为：

============================================================
13. 粗粒化变换 𝓡
============================================================

定义粗粒化变换：

    𝓡 : 𝔠_4 → 𝔠'_4

其中：

    Π → Π'
    G_B → G'_B
    Vol_B → Vol'_B
    W^s → W'^s
    W^c → W'^c
    K_B → K'_B
    S_eff → S'_eff

粗粒化步骤包括：

1. 块合并：

    B'_A = ∪_{a∈A} B_a

2. 体积合并：

    Vol'(B'_A)=Σ_{a∈A}Vol(B_a)

3. 边权重重整：

    W'^s_{AB}
    =
    F_s({W^s_{ab}:a∈A,b∈B})

    W'^c_{AB}
    =
    F_c({W^c_{ab}:a∈A,b∈B})

4. 场变量重定义：

    Φ'_A
    =
    1/Vol'(B'_A)
    Σ_{a∈A} Vol(B_a) Φ_a

5. 作用量参数重整：

    m, λ, c_s, g_U, g_I, g_G
    →
    m', λ', c_s', g_U', g_I', g_G'

裁决原则：

    粗粒化不是任意降采样；
    它必须保持因果方向、局域性、体积加性和谱稳定性。

============================================================
14. 有效参数流
============================================================

设第四编作用量包含参数集合：

    θ_eff
    =
    (m^2, λ, c_s^2, g_U, g_I, g_G, ...)

粗粒化一次后：

    θ_eff → θ'_eff

定义流：

    β(θ_eff)
    =
    θ'_eff - θ_eff

若引入尺度因子 b，可写：

    β_i
    =
    dθ_i / d log b

第五编 R1.0 不要求解析计算 β，只要求记录：

    参数是否趋向稳定；
    是否出现固定点；
    是否出现周期流；
    是否失控发散。

定义固定点候选：

    θ_eff^*

满足：

    𝓡(θ_eff^*) ≈ θ_eff^*

固定点候选通过条件：

1. 几何维数代理稳定；
2. 谱维数稳定；
3. 低谱结构稳定；
4. 有效参数不发散；
5. 标签同构无关。

============================================================
15. 谱流
============================================================

对每个粗粒化层级 n，得到场算符：

    K_B^{(n)}

其谱为：

    Spec(K_B^{(n)})
    =
    {λ_0^{(n)}, λ_1^{(n)}, ...}

定义低谱窗口：

    Λ_low^{(n)}
    =
    {λ_i^{(n)} : λ_i^{(n)} ≤ Λ_cut^{(n)}}

定义谱流距离：

    Dist_spec(n,n+1)
    =
    Dist(
      Normalize(Λ_low^{(n)}),
      Normalize(Λ_low^{(n+1)})
    )

若 Dist_spec 在多个粗粒化步后趋小，则称低谱稳定。

裁决原则：

    低谱稳定是有效模存在的前置条件；
    低谱不稳定时，不能谈粒子候选。

============================================================
16. 谱维数流
============================================================

第四编定义谱维数代理：

    P(τ)
    =
    1/N Tr exp(-τ L_B)

    d_spec(τ)
    =
    -2 d log P(τ) / d log τ

在粗粒化层级 n：

    d_spec^{(n)}(τ)

定义谱维数流差异：

    Dist_dspec(n,n+1)
    =
    Avg_{τ∈T_window}
    |d_spec^{(n)}(τ)-d_spec^{(n+1)}(τ)|

若存在 τ 窗口使得：

    d_spec^{(n)}(τ)

随 n 近似不变，则称存在谱维数平台。

谱维数平台比单次维数估计更强，因为它要求：

    几何与算符在粗粒化下共同稳定。

============================================================
17. 有效模
============================================================

给定场算符：

    K_B ψ_i = λ_i ψ_i

定义模 ψ_i 的支撑体积：

    Supp_eff(ψ_i)
    =
    1 / Σ_a |ψ_i(a)|^4

这是离散版参与比。

若 Supp_eff 很小，则 ψ_i 是局域尖峰模；
若 Supp_eff 与系统规模同阶，则 ψ_i 是延展模。

有效低能模候选应满足：

1. λ_i 位于低谱窗口；
2. ψ_i 在粗粒化下可追踪；
3. ψ_i 不是纯局域缺陷模；
4. ψ_i 对切片和参数扰动稳定；
5. ψ_i 的传播核有可解释的大尺度行为。

定义模稳定性：

    Stab_mode(ψ_i^{(n)},ψ_j^{(n+1)})
    =
    |⟨ψ_i^{(n)}, Lift(ψ_j^{(n+1)})⟩|^2

其中 Lift 是从粗粒层回到细粒层的嵌入或延拓映射。

若某个模序列在多层粗粒化下保持高 Stab_mode，则称为有效模。

============================================================
18. 低能窗口
============================================================

定义低能窗口不是靠外部能量单位，而是靠谱尺度分离。

若谱满足：

    λ_0 ≤ λ_1 ≤ ... ≤ λ_N

存在 k，使得：

    λ_k << λ_{k+1}

或在归一化谱中存在明显 gap，则定义：

    Low = {0,1,...,k}

若无谱隙，可用热核窗口或粗粒化稳定窗口定义低能模。

定义低能投影：

    P_low
    =
    Σ_{i∈Low} |ψ_i⟩⟨ψ_i|

低能有效场：

    Φ_low
    =
    P_low Φ

通过条件：

    P_low 在切片、参数、粗粒化扰动下稳定。

============================================================
19. 有效传播核
============================================================

固定几何上传播子：

    G = K_B^{-1}

低能传播核：

    G_low
    =
    P_low K_B^{-1} P_low

若 K_B 有零模，则使用伪逆：

    G_low
    =
    P_low K_B^+ P_low

有效传播核应检查：

1. 是否随图距离衰减；
2. 是否随块级因果关系呈现方向性；
3. 是否对粗粒化稳定；
4. 是否与维数平台一致；
5. 是否不存在非来源长程异常。

若传播核在大尺度上稳定，才可以作为粒子候选前的传播结构。

============================================================
20. 从有效模到粒子候选的最小判据
============================================================

第五编 R1.0 不定义粒子，但可以定义粒子候选前置判据。

某一组有效模可进入粒子候选讨论，当且仅当：

P_mode1. 低谱稳定：

    λ_i 在粗粒化下稳定。

P_mode2. 模形稳定：

    ψ_i 在切片、参数与粗粒化下可追踪。

P_mode3. 延展性通过：

    ψ_i 不是纯局域缺陷模。

P_mode4. 传播核稳定：

    G_low 在大尺度上稳定。

P_mode5. 近似对称性出现：

    存在近似平移、旋转或其他结构对称性，
    使模可以分组成可解释的多重态。

P_mode6. 有效色散关系可定义：

    模标签与传播尺度之间存在稳定关系，
    类似但不预设 E(p)。

只有 P_mode1-P_mode6 通过，后续第六编才能讨论粒子候选。

裁决原则：

    粒子不是场的任意本征模；
    粒子候选必须是稳定谱、近似对称性和传播结构共同出现的结果。

============================================================
21. 近似对称性
============================================================

在无背景离散结构中，对称性不能预设为连续群。

定义结构近似自同构：

    π: Π → Π

若 π 近似保持：

    G_B, ≺_B, Vol_B, W^s, W^c, K_B

即：

    Dist(𝔠_4,π(𝔠_4)) ≤ ε

则 π 是 ε-近似自同构。

近似对称群候选：

    Aut_ε(𝔠_4)

有效模在 Aut_ε 下的变换若形成稳定多重态，则可以进一步讨论自旋、动量或内部量子数的候选。

第五编 R1.0 只定义 Aut_ε，不直接解释为洛伦兹群或规范群。

============================================================
22. 固定点附近的普适类
============================================================

若粗粒化流存在固定点候选：

    𝔠_4^*

则可定义扰动：

    𝔠_4 = 𝔠_4^* + δ𝔠

线性化粗粒化：

    δ𝔠' = M_R δ𝔠

若 M_R 的特征方向中存在：

    相关方向；
    无关方向；
    边缘方向；

则可定义普适类候选。

裁决原则：

    普适类不是连续 QFT 的直接搬用；
    它表示不同离散微观结构在粗粒化后流向相同大尺度接口包。

若两个不同第二编事件结构经第三、四、五编后流向同一固定点类，则它们可视为同一大尺度物理候选。

============================================================
23. 动力学选择原则
============================================================

第五编需要给出动力学选择原则，但不要求唯一。

候选原则 A：最小稳定泛函。

选择使：

    S_geom + S_upd + S_int

较小，且通过所有硬约束的结构。

候选原则 B：路径权重主导。

按：

    exp[-S_eff^E]

或：

    exp[iS_eff^L]

加权所有结构，观察主导鞍点或相干结构。

候选原则 C：粗粒化固定点优先。

选择在 𝓡 下趋向稳定固定点的结构。

候选原则 D：谱稳定优先。

选择低谱、谱维数和传播核最稳定的结构。

正式主线建议：

    R1.0 使用 A 与 D 做验证；
    R1.1 引入 C；
    R2 以后再尝试 B 的完整路径和。

============================================================
24. 鞍点方程
============================================================

在稳定型权重下，主导配置满足：

    δS_eff / δΦ = 0

这就是第四编给出的：

    K_B Φ = VJ

若对几何变量也变分，需要定义离散几何变分：

    δS_eff / δG_B = 0
    δS_eff / δW^s = 0
    δS_eff / δW^c = 0
    δS_eff / δΠ = 0

但这些变分不是连续度规变分。

它们可以通过离散 moves 实现：

1. 加/删空间边；
2. 调整边权；
3. 合并/拆分块；
4. 改变切片选择；
5. 改变粗粒化协议；
6. 改变因果覆盖边权。

定义 move 集合：

    𝓜 = {m_1,m_2,...}

离散鞍点条件：

    Δ_m S_eff ≥ 0

对所有允许 move m。

若是洛伦兹型相位，则鞍点条件为：

    Δ_m S_L ≈ 0

或相位在主导路径附近相干。

============================================================
25. 采样协议
============================================================

稳定型权重可以使用 Markov 型采样。

状态：

    X = (𝔠_4,Φ)

允许 move：

1. 局域场更新：

    Φ_a → Φ_a + δ

2. 块合并/拆分；

3. 边权微调；

4. 空间边局域翻转；

5. 切片局域替换；

6. 因果覆盖边检查与修正；

7. 更新/干涉痕迹局域重汇聚。

接受率候选：

    P_accept
    =
    min(1, exp[-ΔS_eff^E])

并乘以硬约束：

    χ_pass(X')

若 X' 违反无环、局部有限、标签无关或第三编稳定性红线，则：

    P_accept=0

裁决原则：

    采样是验证工具，不是定义本体；
    采样顺序不得进入物理结构。

============================================================
26. 洛伦兹型相位和符号问题
============================================================

若使用：

    exp[iS_eff^L]

则路径和可能高度振荡。

第五编 R1.0 不强行解决符号问题，只列出允许处理方式：

1. 稳定型权重先验证结构；
2. 洛伦兹型相位只在通过结构上计算；
3. 使用鞍点/相干相位近似；
4. 使用因果顺序限制减少路径空间；
5. 使用粗粒化后有效相位；
6. 暂不做 Wick rotation，除非给出离散因果结构下的明确定义。

裁决原则：

    不能未经说明把 exp(iS) 换成 exp(-S)；
    二者对应不同对象。

============================================================
27. 与因果集动力学的关系
============================================================

传统因果集动力学常讨论：

    顺序生长；
    路径和；
    标号不变性；
    局部有限偏序；
    因果矩阵或 Benincasa-Dowker 型作用量等。

BICFT 第五编的差异在于：

    它不直接从任意偏序求和开始，
    而是从第二编的 Ready/ApplySet/事件痕迹导出的偏序开始；
    再经过第三编空间承载层稳定性；
    再经过第四编块级场与作用量；
    最后才进入第五编权重与路径和。

因此 BICFT 路径和不是：

    直接对所有偏序求和

而是更受约束的：

    对可由局域更新规则形成、
    可构造稳定几何、
    可承载稳定场算符的结构求和。

这可减少任意偏序爆炸问题，但也带来新的任务：

    Ready/ApplySet 与 Resolve 的规则必须足够清楚；
    结构筛选不能偷渡背景；
    路径测度必须同构无关。

28. 最小玩具例：固定一维链上的高斯场
============================================================

取第三编和第四编玩具例的块级一维链：

    B_1 ~ B_2 ~ ... ~ B_N

体积：

    V_a=1

空间权重：

    W^s_{a,a+1}=1

若暂不考虑因果方向，只取稳定型空间场算符：

    K = -c_s^2 Δ_s + m^2 I

则固定几何配分泛函为：

    Z_2
    =
    C(det K)^(-1/2)

本征模近似为离散正弦/余弦模。

当 N 增大、粗粒化保持链结构时，低谱模可稳定追踪。

该玩具例验证：

    固定几何场路径和；
    K_B 谱；
    低模追踪；
    粗粒化谱稳定；

但它不验证：

    完整事件结构路径和；
    粒子谱；
    洛伦兹不变性。

============================================================
29. 最小玩具例：因果链层上的离散波
============================================================

若有层级 l 与空间块 a：

    Φ_a^l

洛伦兹型二次作用量：

    S_L
    =
    1/2 Σ_{l,a}(Φ_a^{l+1}-Φ_a^l)^2
    -
    c_s^2/2 Σ_{l,a}(Φ_{a+1}^l-Φ_a^l)^2
    -
    m^2/2 Σ_{l,a}(Φ_a^l)^2

离散场方程：

    Φ_a^{l+1}-2Φ_a^l+Φ_a^{l-1}
    -
    c_s^2(Φ_{a+1}^l-2Φ_a^l+Φ_{a-1}^l)
    +
    m^2 Φ_a^l
    =
    0

平面波试探：

    Φ_a^l = exp[i(ωl-pa)]

给出离散色散关系：

    4 sin^2(ω/2)
    =
    4c_s^2 sin^2(p/2)
    +
    m^2

小 p,ω 极限：

    ω^2 ≈ c_s^2 p^2 + m^2

该结果只在规则玩具结构上成立。

第五编不能从这个玩具例直接宣布真实粒子存在，只能说明：

    若 BICFT 粗粒化结构接近规则层状图，
    则可出现类似波动模与有效色散关系。

30. 失败模式
============================================================

F1. 测度失败。

    未说明对什么对象求和，或按标签重复计数。

F2. 权重失败。

    exp(-S) 与 exp(iS) 混用，物理意义不清。

F3. 结构路径和过宽失败。

    直接对所有偏序求和，丢失第二编 Ready/ApplySet 来源约束。

F4. 粗粒化失败。

    参数流发散，维数平台消失，谱不稳定。

F5. 低模失败。

    低谱模是局域缺陷模，不是延展有效模。

F6. 近似对称失败。

    没有稳定近似自同构，无法定义动量、自旋或多重态候选。

F7. 粒子偷渡失败。

    未经过谱稳定和色散关系，直接把本征模叫粒子。

F8. 连续偷渡失败。

    直接使用连续路径积分符号 ∫𝒟g，并声称已经无背景推导。

F9. 采样顺序偷渡失败。

    Markov 采样顺序或候选编号影响最终物理结构。
